This is a project I started just to illustrate an understanding of XML. Initially, I created an XML document with a list of restaurant menu items, but I felt it wasn’t very impressive by itself, so I started building a little logic around making it filterable. I started with the data just appended to the bottom of the HTML file, and hidden, but felt that was just phoning it in. I then broke it out to a separate file and used Ajax to import the data. Using constructors, the script will import the xml and instantiate an array of objects defining each restaurant’s menu, with nested arrays of meal-times etc. The project was designed to accept any XML data that has been formatted correctly, with nothing hard-coded into the JS, so that in the future, I can set up more menus and the script will be able to format it successfully.
